The exploration of permanently shadowed regions (PSRs) on the Moon is critical for advancing
our understanding of lunar resources, particularly water ice, and other volatiles. Chandrayaan-2's
imaging instruments have provided valuable datasets, but enhancing the quality of images
captured in these low-illumination regions poses significant challenges. This project focuses on
utilizing Python for image enhancement, applying advanced computational techniques to analyze
and improve PSR images obtained by Chandrayaan-2.
In this work, Python's robust libraries, including OpenCV, NumPy, and Matplotlib, are
leveraged to process and enhance lunar PSR images. Techniques such as histogram
equalization, noise reduction, and contrast adjustment are applied to improve image clarity.
Machine learning algorithms, including deep learning models, are integrated to automate feature
extraction and analyze surface characteristics. These enhancements aim to facilitate better
visualization and interpretation of critical lunar features, enabling improved scientific insights.
The outcomes of this project will contribute to lunar science by providing more precise visual
data for researchers and scientists. Enhanced PSR images will support future mission planning,
resource identification, and exploration strategies. By demonstrating the effective application
of Python in this domain, the project also emphasizes the potential of open-source tools in
advancing space exploration and image processing technologies.
